# BoothHub – Booking Management System for Small Businesses

## Project Idea
BoothHub is a web-based system that helps small business owners manage booking requests, organize schedules, and track orders efficiently.

## Problem
Small businesses face challenges in:
- Managing bookings manually  
- Avoiding scheduling conflicts  
- Organizing customer data  

## Solution
BoothHub provides a simple interface that allows business owners to:
- Manage bookings in one place  
- Organize and track orders  
- Reduce scheduling conflicts  

## Core Features

### Smart Booking System
Users select a service, date, and time.  
The system prevents double booking and shows availability status.

### Customer Data
Stores customer name, phone number, and booking details.

### Dashboard
Displays key information:
- Total bookings  
- Pending requests  
- Monthly activity  

### Manage Services
Admin can:
- Add services  
- Edit services  
- Delete services  

### Manage Orders
Displays:
- Customer information  
- Service  
- Date and time  
- Status  

### Booking Action
Allows admin to:
- Accept bookings  
- Reject bookings  

## Pages
- Login Page  
- Dashboard  
- Manage Services  
- Manage Orders  
- Booking Action
